# Weather Data Visualizer – anupam

A Python project for analyzing, cleaning, and visualizing weather data using Pandas, NumPy, and Matplotlib.  
This project explores temperature, humidity, and rainfall patterns through statistical methods and visual charts.

---

## Project Overview

This project performs a complete weather-data analysis workflow:

- Data acquisition  
- Data cleaning and preprocessing  
- Statistical analysis using NumPy and Pandas  
- Data visualization using Matplotlib  
- Grouping and aggregation  
- Exporting outputs such as cleaned CSV files and PNG images  
- Final summarized report

## Dataset Description

The dataset contains daily weather observations with the following fields:

- date  
- temperature (°C)  
- humidity (%)  
- rainfall (mm)  

Additional optional fields (if included):

- wind_speed  
- pressure  
- cloud_cover
